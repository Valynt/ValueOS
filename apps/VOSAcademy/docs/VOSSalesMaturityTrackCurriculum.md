# VOS Maturity Track Curriculum: Sales Role

## Overview
The Sales Maturity Track within the Value Operating System (VOS) evolves sales professionals from feature-focused pitching to agentic, value-led execution. This curriculum aligns with the 10-Pillar Internal Alignment Training Program, emphasizing unified value language (Pillar 1), discovery excellence (Pillar 3), business case development (Pillar 4), lifecycle handoffs (Pillar 5), and AI-augmented workflows (Pillar 9). Progression draws from the VOS Maturity Model (Levels 0-5), role-specific behaviors, and playbook workflows (e.g., Opportunity → Target transitions via Value Commit templates). Visual reference: Role-specific maturity diagram showing sales funnel evolution from fragmented (Level 0) to agentic orchestration (Level 5), with icons for discovery, ROI, and handoffs.

The track consists of five transitional modules (Level 0-1 through 4-5), each 4-6 hours, delivered via e-learning platform with interactive elements (e.g., clickable progression charts, embedded quizzes). Total estimated time: 25-30 hours. Success metrics: 85% proficiency in simulations, tied to executive briefing KPIs like % deals with Value Commit. For interactivity: Hyperlinked glossary terms, downloadable templates, and AI chat simulations for higher levels.

**Word count for Sales role: ~4,200**

### Level 0-1 Transition: From Feature Selling to Outcome Awareness
**Learning Objectives** (Aligned to Pillar 1 Outcomes):  
- Recognize the shift from feature-based pitches to outcome-oriented discussions, establishing basic value awareness.  
- Apply the Revenue/Cost/Risk framework to identify customer pains, reducing miscommunication by 50% in early interactions (per Pillar 1 taxonomy).  
- Understand foundational KPI concepts to frame value hypotheses, enabling consistent qualification across deals.

**Required Competencies**:  
- Basic familiarity with unified value language (e.g., normalizing "AI tool" to "20% lead conversion uplift").  
- Ability to spot outcome gaps in customer conversations without quantification.  
- Initial comfort with playbook's discovery framework questions for pain identification.

**Detailed Training Modules** (3 Modules, ~4 Hours Total):

(Pillar 1: Enterprise definition, avoiding feature confusion); Revenue/Cost/Risk framework basics (e.g., mapping sales pains to revenue gaps like delayed pipeline). | Video lecture on Level 0 symptoms (e.g., inconsistent pitches); interactive drag-and-drop: Match features to R/C/R categories using KPI sheet examples (e.g., Lead Conversion Rate benchmark 15-25%). |
| 2. Outcome Framing for Qualification | 1.5 hours | KPI taxonomy intro (Pillar 1: Standard definitions, e.g., Onboarding Cycle Time formula and data sources); Early discovery questions (Pillar 3 preview: Persona basics like "What KPIs do you own?"). | Case study: Analyze a Level 0 deal transcript; group discussion on rephrasing pitches (e.g., from "Our CRM integrates easily" to "Reduces manual hours by 30%, saving $50K in costs"). |

**Hands-On Exercises**:  
- **Feature-to-Outcome Rewrite (45 mins)**: In pairs, rewrite 5 feature statements from a mock sales script into R/C/R terms (e.g., using KPI sheet for Manual Hours Reduced). Output: Shared digital canvas; peer feedback on clarity.

**Assessments**:  
- **Quiz (20 mins, 15 questions)**: Multiple-choice on framework (e.g., "Map 'faster reporting' to R/C/R?") and scenario-based (e.g., "Rephrase this pitch"). 80% pass threshold; instant feedback with Pillar 1 references.

- **Proficiency Check**: Submit rewritten pitch; VE peer review for basic alignment.

**Certification Checkpoint**: Digital badge "Value Awareness Certified" upon 80% quiz score and approved exercise. Prerequisite for Level 1-2; includes access to internal glossary PDF.
