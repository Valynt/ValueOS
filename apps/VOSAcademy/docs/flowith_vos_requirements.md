### Program Completion Certification

**Final Simulation Tasks:**
- Build a Value Tree for a given scenario
- Perform a simulated discovery session with an AI persona
- Produce a defensible business case and ROI model
- Create a complete Value Realization Plan
- Build an expansion model for a QBR

**Scoring Rubric:**
- 40% Technical Execution: Correctly builds models, uses templates, quantifies pain
- 30% Cross-Functional Alignment: Completes handoffs, communicates value consistently
- 30% AI Augmentation: Leverages AI tools for discovery prep, analysis, content generation

**Certification Tiers:**
- **Bronze**: Passed all knowledge checks and individual exercises
- **Silver**: Passed final simulation with 80%+ score
- **Gold**: Passed final simulation with 95%+ score and demonstrated exceptional strategic insight

### VOS Maturity Model (6 Levels)

**Level 0: Value Chaos**
- Core Focus: No alignment
- Organizational Behavior: Reactive, inconsistent, fragmented
- Characteristics:
  - No unified value language
  - Sales, CS, Product all speak different "Value dialects"
  - ROI is reactive, inconsistent, or absent
  - Discovery varies widely by representative
  - No lifecycle continuity from Opportunity→Renewal
- Symptoms:
  - Deals driven by features instead of outcomes
  - Value does not survive handoffs
  - CS struggles to prove impact
  - Renewals depend on relationships, not economics
- Actions to Advance:
  - Introduce unified value definitions
  - Begin building a simple value lexicon
  - Identify champions

**Level 1: Value Awareness**
- Core Focus: Intro to value thinking
- Organizational Behavior: Early consistency, basic language
- Characteristics:
  - Teams understand the importance of value
  - Early attempts at discovery standardization
  - Occasional use of ROI during late-stage selling
  - Some KPI language emerging but not consistent
- Strengths:
  - Leadership interest in value
  - Pockets of excellence within Sales or CS
- Actions to Advance:
  - Train on Value Language 101 (Revenue/Cost/Risk)
  - Standardize discovery questions
  - Publish KPI definitions
