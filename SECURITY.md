# Security नीति / Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in ValueOS, please report it privately so we can address it promptly.

- **Email**: security@valueos.com
- **Expected response time**: within 2 business days

Please include:

- A description of the issue and potential impact
- Steps to reproduce or a proof of concept
- Any relevant logs or screenshots

Do **not** open a public GitHub issue for security vulnerabilities.

## Scope

This policy applies to:

- The ValueOS web application
- Backend services and APIs
- Infrastructure-as-code and deployment workflows
- CI/CD automation and scripts

## Disclosure Process

1. **Acknowledgement** within 2 business days.
2. **Assessment** to confirm impact and severity.
3. **Remediation** with a target fix timeline communicated to the reporter.
4. **Coordinated disclosure** once a fix is available.

Thank you for helping keep ValueOS secure.
