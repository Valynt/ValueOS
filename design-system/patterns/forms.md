# Form Patterns

## Input Layout

- Stack labels vertically above inputs.
- Use `--space-2` (16px) between stacked fields.
- Use `--space-4` (32px) between logical form groups.

## Validation

- **Inline Error**: Text appears below the input in `--color-error` (Red).
- **Field Border**: Input border changes to Red.
- **Summary**: For long forms, show a summary alert at the top.

## Actions

- Primary Action (Save/Submit) on the **Right** (desktop) or **Full Width** (mobile).
- Cancel/Secondary Action on the **Left** of the Primary button.
