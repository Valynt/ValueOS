# Sample Test Artifacts

This folder contains intentionally minimal examples of test artifacts for reference in documentation.

- `playwright-report-sample.html` is a tiny HTML stub showing the kind of summary page Playwright generates.
- `last-run.sample.json` mirrors the structure of Playwright's `.last-run.json` file with placeholder values.

Actual test output is produced in `playwright-report/` and `test-results/` during local runs and is ignored by Git.
