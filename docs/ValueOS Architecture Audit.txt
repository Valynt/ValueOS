ValueOS Architecture Audit
A) Executive Map
What This Platform Is
ValueOS is a multi-tenant, agent-orchestrated B2B value-selling platform. It guides enterprise sales teams through a structured "value lifecycle" (Opportunity → Target → Realization → Expansion) where specialized AI agents analyze companies, build financial models, map value trees, and generate narratives — all rendered through a Server-Driven UI (SDUI) system that agents can dynamically compose and mutate. The platform sits on Supabase/Postgres with Kafka eventing, integrates with CRMs (Salesforce, HubSpot), uses MCP for LLM tool access, and bills via Stripe.

Primary User Outcomes
Discover and qualify opportunities — AI-assisted company research, 10-K analysis, competitive intelligence
Build value cases — Financial modeling with KPIs, assumptions, scenarios, ROI calculations
Map value architectures — System mapping, causal loops, intervention design
Track realization — Monitor actual vs. predicted value delivery with feedback loops
Expand accounts — Identify expansion opportunities within existing customers
Maintain integrity — Zero-hallucination guarantees via tiered ground truth resolution (EDGAR/XBRL → Market Data → Benchmarks)

What This Platform Is
ValueOS is a multi-tenant, agent-orchestrated value-selling platform that helps B2B sales teams quantify, model, and track the business value of their solutions for prospects and customers. The data layer is Supabase/Postgres with Kafka eventing, Stripe billing, and CRM integrations (Salesforce, HubSpot).

Build value models — KPI definition, financial modeling, assumption management, scenario analysis
Generate business cases — Agent-produced ROI narratives, impact cascades, value trees
Track value realization — Post-sale metric tracking, feedback loops, confidence scoring
Expand accounts — Identify upsell/cross-sell opportunities based on realized value
Maintain data integrity — Ground truth verification against EDGAR/XBRL, hallucination detection, integrity veto

ValueOS Architecture Audit
A) Executive Map
What This Platform Is
ValueOS is a multi-tenant, agent-orchestrated B2B value-selling platform. It guides enterprise sales teams through a structured "value lifecycle" (Opportunity → Target → Realization → Expansion) using AI agents that research companies, model financial impact, generate narratives, and track realized value. The UI is server-driven (SDUI): agents produce JSON schemas that the frontend renders dynamically. The data layer is Supabase/Postgres with RLS-based tenant isolation, Kafka for eventing, and a tiered "Ground Truth" engine (EDGAR/XBRL/market data) to prevent financial hallucinations.

Primary User Outcomes
Discover and qualify opportunities — AI-assisted company research, 10-K analysis, competitive intelligence
Build value cases — Financial modeling with KPIs, assumptions, scenarios, ROI calculations
Generate buyer-facing narratives — Agent-produced business cases, impact cascades, executive summaries
Track realized value — Post-sale value realization monitoring with feedback loops
Expand accounts — Identify expansion opportunities within existing customers
Manage multi-tenant access — Org-scoped RBAC, billing, API keys, audit trails

4. Interaction & Behavioral Logic
01
View Switching

Navigation updates use an "Active State" class switching system. When a tab is selected, the corresponding ID in the `main-view-container` is set to `display: block` with a CSS `translateY` + `opacity` animation.

02
Agent Stream Trigger

Submitting a prompt triggers a dual action: (1) The Right Sidebar slides in via `translate-x-0`, and (2) The `agent-thinking` state is activated, showing a pulse animation on three horizontal bars representing data ingestion.

03
Chat Bubble Alignment

User bubbles are `flex-row-reverse` with a black background (#000). Agent bubbles are standard row alignment with a white background and 1px border. Radius is set to 1.5rem (24px) for all bubbles.
