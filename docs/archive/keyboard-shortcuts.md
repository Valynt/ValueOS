# Keyboard Shortcuts

ValueOS provides contextual keyboard shortcuts that adapt based on your current workflow stage, allowing power users to work faster without leaving the keyboard.

## How to View Shortcuts

Press `⌘/` (Command + /) at any time to see all available shortcuts for your current stage.

---

## Global Shortcuts

These shortcuts work everywhere in ValueOS:

| Shortcut | Action | Description |
|----------|--------|-------------|
| `⌘K` | Command Palette | Open the command palette to search for any action |
| `⌘\` | Toggle Focus Mode | Hide chat interface and focus on canvas |
| `⌘/` | Show Shortcuts | Display this shortcuts help dialog |

---

## Tips for Power Users

### 1. Learn Stage-Specific Shortcuts

Focus on learning the shortcuts for the stage you use most. The shortcuts are designed to match the natural workflow progression.

### 2. Use Focus Mode

Press `⌘\` to enter Focus Mode, which hides the chat interface and gives you more canvas space. This is especially useful when reviewing complex models or system maps.

### 3. Combine with Command Palette

Use `⌘K` to access any action quickly. The command palette searches across all available actions and shows their keyboard shortcuts.

### 4. Hover for Hints

Hover over any button to see its keyboard shortcut displayed as a tooltip.

## Troubleshooting

### Shortcut Not Working?

1. **Check your current stage** - Some shortcuts only work in specific stages
2. **Ensure focus is on ValueOS** - Shortcuts won't work if another app has focus
3. **Check for conflicts** - Browser extensions may override shortcuts
4. **Try the command palette** - Use `⌘K` as a fallback

### Browser Extension Conflicts

Some browser extensions use the same shortcuts. Common conflicts:

- `⌘K` - Used by many search extensions
- `⌘/` - Used by some help extensions

If you experience conflicts, try disabling extensions or using the command palette instead.
